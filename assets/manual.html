How does it work:
Well, its quite simple really: Caesura monitors your phone's SMS (a.k.a. text messages) in the background, waiting for a command. To control a phone, install the Caesura app on the phone that you want to control (yes, it MUST be on the phone that is being controlled, not yours) and check the 'Enable Service' checkbox on the main Caesura screen. When you want to control the phone in question, all that you have to do is text !login to that phone, or, if you are a lazy person, install the app on your phone too and click the 'Login to Another Phone' button on the main Caesura screen. After a quick moment or two, the phone will respond and tell you that it is ready for a command. From that moment on, all SMS messages are treated as commands until you send the exit command. For a full list of commands, scroll to the bottom of this document DO NOT FORGET TO SEND THE EXIT COMMAND OR ELSE OTHER PEOPLE MAY BE ABLE TO CONTROL PHONE

Controlling your own phone: To control your own phone remotely, borrow a friend's phone and send a text message containing !login to your phone. After your phone responds, enter your password and your phone will send a text saying that Caesura is logged in. From this moment on, you can send any command as a text message. For a full list of commands, scroll to the bottom of this document.

Controlling someone else's phone: Before you consider controlling someone else's phone, consider this: by controlling a phone that belongs to another person, you can play innocent practical jokes, however, this app also has the power to read/send sms messages, dial phone numbers, read contacts, etc. Controlling another person's phone (aside from your own child's phone, which is legal in some countries) may be illegal. Check your country's laws before use. Alright, now that the "cover my butt" clause is out of the way: to control another phone, install this app on the phone that you want to control. The phone being controlled must be an Android 2.1 or newer device. However, you can control that phone from any phone (yes, even an iPhone) since commands are sent by text message.

Using commands: Once you have logged into Caesura remotely from another phone, any text that you send will be treated as a command until you send the exit command. To send a command, type it into a text followed by a space and any additional parameters. For example, the toast command which is listed in the commands list as: toast [ticker] [text] accepts two additonal parameters, ticker and text (for more information on what these two parameters do, read the commands list below). To use this command enter the command name, a space, the ticker, another space, and then the text. For example:

toast [ticker_text] [text] : toast ExampleNotification Hello, this is the notification message

Notice, although the different parameters are separated by spaces, the last parameter, which is meant to accept a full sentence of text, can have spaces. Some more examples are below, along with the way that they are listed in the command list:

sendsms [phone_number] [text] : sendsms 9375551919 Hello, this is a text message sent from a phone.
call [phone_number] [additional_options] : call 9375551919 speaker
help [command] : help torch

Commands List: !login text this command to phone to connect and start controlling the phone. After you are connected, you can use any of the below commands to interact until you are done. When finished, use the exit command to logout. DO NOT FORGET TO LOG OUT.

  1. toast [ticker] [text] Displays a notification on the Android Notification Bar. Replace [ticker] With the title for the notification (with no spaces and replace [text] with the text that will be displayed on the notification (can have spaces).
  2. sendsms [phone_number] [text] Sends a text message. Replace [phone_number] with 10 digit number to send text to. Replace [text] with text to send. Message length is limited to phone character limitation (between 12-155 characters depending on the phone).
  3. torch Send this command to turn on the flashlight. Please note: this command only works with some phones. Remember to turn off the flashlight by sending the torch command again before logging out. Logging out does not turn off flashlight!!
  4. panic If you lose your phone, send this command to turn the ringer volume up all the way, and make the phone ring and flash the light to help you find it. Send again to turn off panic mode.
  5. call [phone_number] [options] Replace [phone_number] with a number to call. For example: call 9875551313 will make the phone call that number. Optionally, add the word speaker to the end where [options] is to make the phone call on speaker phone, like so: call 9875551313 speaker.
  6. help [command] Replace [command] with one of the commands from this list and Caesura will tell you what the command does and how to use it.
  7. status Tells info about phone, including battery level, and whether or not wifi and gps are on.
  8. password [new_password] Sets the login password for Caesura to [new_password]. For example, password smelly_cheese sets the password to smelly_cheese
  9. unlock Bypasses the phones keyguard. If the phone has a password of PIN code set, it will disable it temporarily or until the phone is rebooted.
  10. block [phone_number] Adds [phone_number] to the block list, making the phone being controlled unable to get text messages from that number. However, calls will still go through.
  11. unblock [phone_number] Removes [phone_number] from the block list.
  12. blocked Retrieves a list of numbers that were blocked with the block command.
  13. forward [phone_number] Adds [phone_number] to the forward list, making any text received from that number be automatically sent to the phone specified with forwardto command.
  14. unforward [phone_number] Removes [phone_number] from the forward list.
  15. forwarded Returns a list of all the numbers currently being forwarded
  16. forwardto [phone_number] Sets the phone number that all texts from numbers on the forward list will be sent to.
  17. contacts [command] [name] Allows interaction with the contacts database. Replace command with either search, or getphone. To search for all contacts with the name Lori, for example: contacts search lori
    or to get the phone number of a lady named Sarah Stuart: contacts getphone Sarah Stuart
  18. more Whenever a Caesura message ends with ...more it means that there is more message that would not fit in the text. This command gets the rest of the last response.
  19. speak [text] Makes phone speak out loud. This is a great prank! Replace [text] with words for the phone to say out loud. First time that you send this command, the text WILL NOT be spoken because Caesura first has to start up Text To Speech. The second time you send, it will speak. To insure that the text spoken can be heard well, use this command to turn the volume up first: Volume media 20
  20. volume [channel] [amount] Sets Android system volumes. Replace [channel] with one of the following: media, notifications, alarm, dtmf, ring, sys, and replace amount with either mute, unmute, or a value 0 to 20 to set the new volume. 0 represents minimum volume (off) and 20 represents maximum. The different channels mean different things. Media is music/apps/TextToSpeeech/etc, notifications is SMS ringtone volume, ring is ringtone volume, alarm is alarm volume, dtmf is phone dial tone volumes, and sys is system sounds volume.
  21. speakerphone Toggles speakerphone for calls that are in progress.
  22. Vibrate Toggles phone vibrate settings. DOES NOT MUTE RINGER. To mute ringer, you must first use: volume ring mute
  23. wifi Toggles phone WIFI radio.
  24. gps Toggles phone GPS radio.
  25. about Tells Caesura Version and Build Number.
  26. list Lists supported Caesura commands in no particular order.
  27. exit Logs out of Caesura. DO NOT FORGET TO SEND THIS COMMAND WHEN FINISHED.
