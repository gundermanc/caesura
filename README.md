INTRO:

This is an old Android App project from high school that allows a user
to remote control his or her phone via SMS message.

FEATURES:
 - Remotely Send Text
 - Remotely Recv. Text
 - Toggle flashlight
 - Send Notifications
 - Toggle Wifi, bluetooth, vibrate.
 - Set volume
 - Call numbers
 - Speakerphone
 - Much more.
